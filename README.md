# ✈️ Airline Price Prediction

## 📌 Overview

This project predicts airline ticket prices based on features like airline, source, destination, duration, and number of stops using machine learning.

---

## ⚙️ Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib / Seaborn

---

## 🧠 Project Workflow

1. Data collection
2. Data cleaning & preprocessing
3. Feature engineering
4. Model training (Regression)
5. Model evaluation
6. Prediction

---

## 📁 Project Structure

```bash
airline-price-prediction/
│
├── train.py                # Model training
├── preprocessing.py        # Data cleaning
├── model.py                # Model building
├── app.py (optional UI)
├── requirements.txt
│
├── data/
├── images/
```

---

## 🚀 How to Run

pip install -r requirements.txt
python train.py
